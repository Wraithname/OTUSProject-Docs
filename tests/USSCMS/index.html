    12.0.0.2 Негативный сценарий¶

    Шаг Описание шага Ожидаемый результат
    1 Перейти на страницу "Отчеты" Отображается страница отчетов
    2 Нажать кнопку "Обновить данные" Появляется всплывающее сообщение с текстом "Пожалуйста подождите, идет формирование отчета"
    3 Прервать процесс формирования отчета (например, закрыть браузер) Отчет не формируется, и на странице отображается сообщение об ошибке "Ошибка при формировании отчета"
    4 Перейти на страницу "Отчеты" Отображается сообщение "Нет доступных данных для отчета"

    12.0.0.3 Нейтральный сценарий¶

    Шаг Описание шага Ожидаемый результат
    1 Перейти на страницу "Отчеты" Отображается страница отчетов
    2 Нажать кнопку "Обновить данные" Появляется всплывающее сообщение с текстом "Пожалуйста подождите, идет формирование отчета"
    3 Подождать менее 15 минут и снова зайти на страницу "Отчеты" На странице отображается сообщение "Отчет еще формируется, пожалуйста, подождите"
    4 Подождать 15 минут и снова зайти на страницу "Отчеты" На форме отображается сформированный отчет
    5 Проверить, что отчет содержит структурированный список проектов Отчет отображает список проектов, но может не содержать информации для всех модулей

    12.0.0.4 Основной сценарий¶

    Шаг Описание шага Ожидаемый результат

    4 Проверить, что информация по объектам отображается корректно Отображается информация по объектам, из которых состоит выбранный модуль

    2 Выбрать продукт из списка Отображается список модулей, входящих в выбранный продукт

    USSCMS-ACUS03

    12.0.0.7 Основной сценарий¶

    Шаг Описание шага Ожидаемый результат
    1 Перейти на страницу "Проекты" Отображается список доступных проектов
    2 Выбрать проект из списка Выбранный проект выделяется, и активируется кнопка "Сформировать отчет"
    3 Нажать кнопку "Сформировать отчет" Появляется диалоговое окно со списком форматов для конвертации отчета
    4 Проверить, что диалоговое окно содержит доступные форматы В диалоговом окне отображается список форматов

    12.0.0.8 Негативный сценарий¶

    Шаг Описание шага Ожидаемый результат
    1 Перейти на страницу "Проекты" Отображается список доступных проектов
    2 Не выбирать проект и нажать кнопку "Сформировать отчет" Появляется сообщение об ошибке "Пожалуйста, выберите проект перед формированием отчета"
    3 Выбрать проект и нажать кнопку "Сформировать отчет" Появляется диалоговое окно со списком форматов
    4 Закрыть диалоговое окно и снова нажать кнопку "Сформировать отчет" Появляется сообщение "Выберите проект для формирования отчета", если проект не был выбран

    12.0.0.9 Нейтральный сценарий¶

    Шаг Описание шага Ожидаемый результат
    1 Перейти на страницу "Проекты" Отображается список доступных проектов
    2 Выбрать проект из списка Выбранный проект выделяется, и активируется кнопка "Сформировать отчет"
    3 Нажать кнопку "Сформировать отчет" Появляется диалоговое окно со списком форматов для конвертации отчета
    4 Выбрать один из форматов и нажать "ОК" Отчет формируется в выбранном формате, и отображается сообщение о завершении процесса
    5 Закрыть диалоговое окно без выбора формата Диалоговое окно закрывается, и процесс формирования отчета не начинается

    Шаг Описание шага Ожидаемый результат

    3 Проверить специальное поле на наличие информации о миграции Специальное поле пустое или отображает сообщение "Информация отсутствует"

    USSCMS-ACUS07

    12.0.0.19 Основной сценарий¶

    Шаг Описание шага Ожидаемый результат
    1 Войти в систему с учетной записью пользователя с необходимыми правами Пользователь успешно входит в систему
    2 Перейти на страницу "Модули" Отображается список доступных модулей
    3 Выбрать модуль из списка Открывается окно с информацией о выбранном модуле
    4 Проверить наличие кнопки "Изменения ответственного" Кнопка "Изменения ответственного" отображается на форме
    5 Нажать на кнопку "Изменения ответственного" Отображается список сотрудников, которые могут быть назначены ответственными
    6 Проверить, что список сотрудников содержит актуальные данные В списке отображаются все сотрудники, имеющие право быть назначенными ответственными
    7 Выбрать сотрудника из списка и подтвердить изменение Ответственный за процесс перевода модуля успешно изменяется на выбранного сотрудника

    12.0.0.20 Негативный сценарий¶

    Шаг Описание шага Ожидаемый результат
    1 Войти в систему с учетной записью пользователя без необходимых прав Пользователь не может войти в систему или не видит возможность изменения ответственного
    2 Перейти на страницу "Модули" Отображается список доступных модулей
    3 Выбрать модуль из списка Открывается окно с информацией о выбранном модуле
    4 Проверить наличие кнопки "Изменения ответственного" Кнопка "Изменения ответственного" отсутствует на форме
    5 Нажать на кнопку "Изменения ответственного" Ничего не происходит или отображается сообщение об ошибке
    6 Проверить, что список сотрудников пустой В списке сотрудников отображается сообщение "Нет доступных сотрудников"

    12.0.0.21 Нейтральный сценарий¶

    Шаг Описание шага Ожидаемый результат
    1 Войти в систему с учетной записью пользователя с необходимыми правами Пользователь успешно входит в систему
